




# Inherit statement

To prevent default activation,
	- if parent_func has parameters -> first statement in child_func is 
								+ `super(arg_list)` to activate
								+ `preventDefault()` to prevent default activate
	- if parent_func has no param   -> first statement is preventDefault()

-> Scope of parent_func is the scope of the function (or param of function)
